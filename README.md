# 🎨 BrandGenie Frontend

> Web interface for BrandGenie - AI-Powered Brand Identity Generator

A sleek, dark-themed web application that connects to the BrandGenie AI agent backend to generate complete brand identities including creative names, taglines, brand stories, and domain recommendations.

[![Live Demo](https://img.shields.io/badge/demo-live-success)](https://brandgenie-frontend.vercel.app)

## 🔗 Backend Repository

This is the **frontend** repository. The backend AI agent is located here:

👉 **[BrandGenie Backend](https://github.com/ALADETAN-IFE/Hng-stage3-be-ai-agent)**

## ✨ Features

- 🎨 Modern dark UI with glassmorphism design
- ⚡ Lightning fast static HTML/CSS/JS
- 📱 Fully responsive design
- 🔌 Real-time API connection to backend
- ✨ Smooth animations and transitions
- 🎯 User-friendly with example prompts
